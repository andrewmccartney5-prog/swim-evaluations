# Swim Lesson Evaluations

A simple, phone-friendly tool for evaluating beginner swimmers (built with 3- and 4-year-olds in mind). Mark each child's strengths and the skills to work on, then generate a warm, ready-to-share note for parents.

Everything runs in the browser — no accounts, no internet needed once it loads. Saved swimmers stay on the device that created them.

## Put it online with GitHub Pages (free)

1. Create a new repository on GitHub (e.g. `swim-evaluations`). Make it **Public**.
2. Upload **`index.html`** (and this README) into the repo. The file must be named exactly `index.html`.
3. Go to the repo's **Settings → Pages**.
4. Under "Build and deployment", set **Source = Deploy from a branch**, **Branch = main**, **Folder = / (root)**, then **Save**.
5. Wait ~1 minute. The page appears at `https://YOUR-USERNAME.github.io/swim-evaluations/`.
6. Open that link on a phone and tap **Share → Add to Home Screen** for a one-tap app icon.

## How she uses it

- Type the swimmer's name (age and she/he are optional — they just make the note read naturally).
- Tap **💪 Strength** or **🎯 Work on** for any of the 30 skills. Tap again to clear.
- **✨ Create evaluation** writes the parent note. **🔁 Reword it** gives a fresh version.
- Edit the note freely, then **📋 Copy** or **🖨️ Print / Save PDF**.
- **💾 Save swimmer** keeps each child in the roster to come back to later.
